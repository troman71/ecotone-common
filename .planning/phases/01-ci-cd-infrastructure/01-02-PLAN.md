    2. Confirm all Plan 01 changes are committed (check git status is clean for tracked files).

    3. Create an annotated tag on the current HEAD:
       ```
       git tag -a v0.1.0 -m "Release v0.1.0: Initial CI/CD infrastructure

       - GitHub Actions CI workflow (lint + test on push/PR)
       - Ruff linting enforcement
       - pytest across Python 3.11, 3.12, 3.13

       ```
       git push origin v0.1.0

       ```
       git show v0.1.0
       ```

    - pyproject.toml version matches tag (both "0.1.0")
  </verify>
